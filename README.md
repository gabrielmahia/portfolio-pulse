# 📊 Portfolio Pulse

**Live portfolio health dashboard for the East Africa AI Stack.**

Inspired by Rundown AI guides:
- "Build a Daily Command Center with Claude Live Artifacts" (April 21, 2026)
- "Build Automated Business Reports with AI" (May 18, 2026)

## What It Shows
- GitHub: total repos, key repo stars/forks
- PyPI: 4 published packages + versions
- Dev.to: article count, reactions, latest title
- HuggingFace: 4 Swahili/Kenya datasets
- Kaggle: 5 research notebooks

## Run It
```bash
pip install streamlit pandas
streamlit run app.py
```
